# Akavarius

## Formål
Dette repository er den autoritative dokumentation for bådprojektet Akavarius og fungerer som bådens digitale tvilling.

## Struktur
Repositoryet er organiseret efter bådens delprojekter og fælles dokumentation.

## Arbejdsgang
Chat bruges til idéudvikling og beslutninger. Når en beslutning er truffet, dokumenteres den i GitHub.

## GitHub
GitHub er den eneste autoritative kilde til projektets dokumentation.

## ChatGPT
ChatGPT anvendes til sparring, projektledelse og vedligeholdelse af dokumentationen.
